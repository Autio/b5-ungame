# The Babylon Project

### A Conversation Game Inspired by Babylon 5

> *"The Babylon Project was our last, best hope for peace... a place where humans and aliens could work out their differences peacefully."*

A browser-based conversation game in the spirit of [The Ungame](https://en.wikipedia.org/wiki/The_Ungame), themed around the philosophical depth of Babylon 5. No winners, no losers—just meaningful conversations.

## Play Now

**[Launch The Babylon Project](https://yourusername.github.io/b5-ungame/)**

## About

This is not a competitive game. It's a structured way to have deeper conversations with friends, family, or anyone willing to explore thoughts and feelings together.

### How to Play

1. **Gather your council** - Add 2-10 players (ambassadors)
2. **Choose your sector** - Select question categories by difficulty/depth
3. **Draw cards** - Take turns drawing and answering questions
4. **Listen deeply** - There are no wrong answers, only sharing
5. **Pass when ready** - Move to the next ambassador

### Question Categories

| Sector | Description |
|--------|-------------|
| **Green Sector** | Light & Fun - Easy conversation starters |
| **Blue Sector** | Reflective - Personal stories and memories |
| **Grey Council** | Deep - Meaningful, thoughtful exploration |
| **Vorlon** | Philosophical - Abstract, big-picture questions |
| **Shadows** | Challenging - Provocative, may cause discomfort |

The **Vorlon** category asks the classic question: *"Who are you?"*
The **Shadows** category asks: *"What do you want?"*

## Features

- 100+ thoughtfully crafted questions
- Beautiful B5-inspired visual design
- Works on desktop and mobile
- No account required
- Session state saved locally
- Keyboard navigation (Space to draw, N for next, S to skip)

## Hosting on GitHub Pages

1. Fork this repository
2. Go to Settings → Pages
3. Set source to "main" branch, root folder
4. Your game will be live at `https://yourusername.github.io/b5-ungame/`

## Local Development

Simply open `index.html` in any modern browser. No build process required.

```bash
# Or use a simple server
npx serve .
# or
python -m http.server 8000
```
